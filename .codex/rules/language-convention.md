# Language Convention

This project serves the Vietnamese real estate market.

- UI text, LLM prompts, and chatbot responses: Vietnamese.
- Code comments and docstrings: English.
- Variable/function/class names: English.
- URL slugs use Vietnamese without diacritics: `/nha-dat-ban`, `/nha-dat-cho-thue`, `/thi-truong`, `/dang-nhap`, `/dang-ky`.
