\section*{Kết quả đạt được}
\addcontentsline{toc}{section}{Kết quả đạt được}

Sau quá trình nghiên cứu và triển khai, đồ án đã đạt được các kết quả chính sau:

\begin{enumerate}
    \item \textbf{Xây dựng thành công nền tảng web bất động sản hoàn chỉnh} với frontend Next.js 16 + React 19 và backend FastAPI, bao gồm các chức năng: duyệt danh sách nhà đất bán/cho thuê, lọc đa tiêu chí, sắp xếp, xem chi tiết bất động sản, dashboard thống kê thị trường, hệ thống xác thực người dùng qua JWT, và bảng điều khiển quản trị (admin dashboard).

    \item \textbf{Phát triển chatbot đa tác tử (Multi-Agent) hoạt động ổn định} với 4 agent chuyên biệt (Property Search, Market Analysis, Legal Advisor, Investment Advisor) và router phân loại ý định. Chatbot có khả năng hiểu ý định người dùng bằng tiếng Việt qua cả keyword-based routing và Gemini-based routing, điều phối đến agent phù hợp, truy xuất dữ liệu qua hybrid search (SQL + pgvector kNN + Cohere rerank + Redis cache) và sinh phản hồi có trích dẫn nguồn rõ ràng.

    \item \textbf{Triển khai kiến trúc chatbot hai lớp} với pipeline nội bộ (backend/app/services/chatbot/) xử lý nhanh cho người dùng cuối và Agent Service (agent\_service/) chạy LangGraph StateGraph riêng biệt cho multi-step reasoning nâng cao. Hệ thống tích hợp cơ chế memory và preferences để ghi nhớ ngữ cảnh người dùng.

    \item \textbf{Xây dựng pipeline thu thập và xử lý dữ liệu tự động} từ batdongsan.com.vn, bao gồm: crawl dữ liệu bằng Playwright với stealth mode cho cả tin bán, tin cho thuê, dự án và tin tức; làm sạch và chuẩn hóa; làm giàu (geocoding qua Nominatim/Goong, trích xuất intent tags); chunking ngữ nghĩa; tạo embedding BGE-M3 (1024 chiều); và nạp vào PostgreSQL + pgvector qua các ingestor module. Pipeline được lập lịch tự động qua Apache Airflow với 4 DAGs.

    \item \textbf{Triển khai hệ thống tìm kiếm lai hiệu quả} kết hợp giữa SQL filtering cho dữ liệu có cấu trúc và pgvector kNN với HNSW index cho tìm kiếm ngữ nghĩa, có reranking qua Cohere và caching qua Redis để cải thiện độ chính xác và tốc độ.

    \item \textbf{Thiết kế kiến trúc microservices có khả năng mở rộng} với Docker Compose, tách biệt 5 service (frontend, backend, agent service, database, cache), hỗ trợ triển khai dễ dàng trên nhiều môi trường.

    \item \textbf{Xây dựng hệ thống observability toàn diện} với agent tracing (AgentTrace, AgentTraceStep, AgentLLMCall, AgentRetrievalEvent), chat feedback, memory proposals, source readiness monitoring và Prometheus metrics endpoint.

    \item \textbf{Bộ test phong phú} với hơn 60 file test bao phủ toàn bộ các module: chatbot pipeline, agent service, hybrid search, RAG, data pipeline, legal KB, crawler, memory/preferences và admin.
\end{enumerate}
