Quy trình xử lý dữ liệu qua các bước:

\begin{enumerate}
    \item \textbf{Crawl:} Playwright thu thập dữ liệu từ batdongsan.com.vn $\rightarrow$ CSV.
    \item \textbf{Clean:} Chuẩn hóa giá (parse ``4,68 tỷ'' $\rightarrow$ 4.68), diện tích, phân loại.
    \item \textbf{Enrich:} Geocode địa chỉ $\rightarrow$ tọa độ (lat/lon), trích xuất intent tags.
    \item \textbf{Chunk:} Chia mỗi listing thành 4 loại chunk (overview, description, location, intent\_tags).
    \item \textbf{Embed:} Tạo vector BGE-M3 1024 chiều cho từng chunk.
    \item \textbf{Load DB:} Upsert vào PostgreSQL (listings) và pgvector (chunks).
    \item \textbf{Serve:} FastAPI truy xuất dữ liệu qua SQL + kNN, frontend hiển thị.
\end{enumerate}
